# 🐍 Snake Game (HTML Single File)

브라우저에서 바로 실행되는 **스네이크 게임(Snake Game)** 입니다.  
**HTML 파일 1개에 CSS + JavaScript를 모두 포함**한 단일 파일 버전이며, **키보드 화살표 키**로 조작합니다.

---

## ✨ 주요 기능

- ✅ **단일 파일 구성**: `snake.html` 하나로 실행 (CSS/JS 내장)
- ✅ **키보드 조작**: 방향키(↑ ↓ ← →)
- ✅ **점수 표시** 및 **최고 점수 저장(localStorage)**
- ✅ **벽/자기 몸 충돌 시 게임 오버**
- ✅ **Space**: 시작/일시정지, **R**: 재시작
- ✅ 먹이를 먹을수록 **속도가 점진적으로 증가**

---

## 🎮 조작 방법

| 키 | 동작 |
|---|---|
| ↑ ↓ ← → | 이동 |
| Space | 시작 / 일시정지 |
| R | 재시작 |

> 방향 전환 시 **역방향(자기 방향 반대로 즉시 전환)은 방지**되도록 처리되어 있습니다.

---

### 1) 로컬에서 실행
1. 이 레포지토리를 다운로드 또는 클론합니다.
